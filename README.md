# Instacart Market Basket Analysis using Power BI
This project analyzes customer purchasing behavior using the Instacart Market Basket Analysis dataset. The dashboard provides insights into order trends, reorder behavior, product performance, and customer purchasing patterns using Power BI.
----
Objectives
Analyze customer purchasing behavior
Identify top-selling products
Measure reorder rates
Discover popular departments and aisles
Track order trends
Build an interactive Power BI dashboard
------
Tools Used
Power BI
Power Query
DAX
Excel
SQL
-------
Dataset
Source:
https://www.kaggle.com/datasets/psparks/instacart-market-basket-analysis
-------
Dashboard Pages
Executive Summary
Customer Behavior
Product Performance
Department & Aisle Analysis
------
Key KPIs
Total Orders
Products Sold
Reorder Rate
Average Basket Size
Average Days Between Orders
Top Products
Top Departments
Top Aisles
